# tzjingtai_core

台州京泰 ERPNext 通用定制 app，作为非专项业务（非承兑汇票）的定制逻辑公共容器。

## 定位

`acceptance` app 专注于承兑汇票全生命周期管理。其它不属于某个垂直业务模块的定制（物料编码规则、通用字段扩展、公共报表、跨模块流程钩子等）统一放在这里，避免 app 数量爆炸。

当某块逻辑膨胀到足以独立成模块时，再从本 app 拆出去。

## 当前功能

- **物料编码按物料组自动生成**：`F(P) 客户产品` 组需用户手填以 `F(P)` 开头的料号；其它分组按 `<前缀>-<序号>` 自动生成，前缀从 item_group 名字中抠取。
